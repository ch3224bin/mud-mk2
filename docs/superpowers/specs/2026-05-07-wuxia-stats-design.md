# 무협 스탯 시스템 설계

**날짜:** 2026-05-07  
**상태:** 승인됨

---

## 개요

기존 D&D 방식의 스탯(STR/DEX/CON/INT/POW/CHA)을 무협 테마에 맞게 전면 재설계한다.  
스탯은 **속성(屬性)** 6개와 **무예(武藝)** 9개로 구성되며, 이를 토대로 생명·내력·체력 파생 스탯이 계산된다.

---

## 속성 (屬性) — 6개

캐릭터의 기본 신체·정신 자질을 나타내는 스탯.

| 속성 | 한자 | 주요 영향 |
|------|------|-----------|
| 근력 | 筋力 | 물리 공격력, 장비 부하 감소 |
| 체질 | 體質 | 생명(HP) 최대치 |
| 민첩 | 敏捷 | 회피, 명중, 연속 공격 기회 |
| 지력 | 智力 | 학식, 명중, 내공·비급 편찬 |
| 의지 | 意志 | 반격, 방어, 부정 효과 저항, 체력(AP) 최대치 |
| 경맥 | 經脈 | 내력(MP) 최대치, 내공 관련 능력 |

### 자질(質) 시스템

- 각 속성에는 **자질**이라는 상한선이 존재한다.
- 캐릭터 생성 시 자질을 **최소 40, 최대 80** 범위에서 속성별로 분배한다.
- 실제 속성 수치는 0에서 시작하여 무공 레벨업을 통해 자질 상한까지 성장한다.
- 자질 80 이상 돌파는 **특별한 무공**을 배워 해당 무공 레벨이 오를 때만 가능하다.

---

## 무예 (武藝) — 9개

무공을 배우고 사용하면서 숙련도가 오르는 스탯.

### 성장 방식

- 특정 무공을 배우면 해당 무예 카테고리의 숙련도를 올릴 수 있게 된다.
- 무공을 사용할수록 해당 무공의 레벨이 상승한다.
- 무공 레벨이 오를 때마다 연관된 **속성 스탯**이 함께 증가한다.
  - 예: 태권도(권장 계열) 레벨 1 증가 → 근력, 권장 스탯 증가

### 무예 목록

| 무예 | 설명 |
|------|------|
| 내공(內功) | 내력(MP)에 기여하는 핵심 무예. 내공이 쌓일수록 내력이 증가한다. |
| 절기(絶技) | 방어력에 영향을 주는 특수 기예. |
| 경공(輕功) | 기동력 및 회피에 영향. |
| 권장(拳掌) | 외공 — 맨손·장법 계열 무기 |
| 검법(劍法) | 외공 — 검 계열 무기 |
| 도법(刀法) | 외공 — 도 계열 무기 |
| 장병(長兵) | 외공 — 창·봉 등 장병기 계열 |
| 기문(奇門) | 외공 — 암기·기문병기 계열 |
| 사술(射術) | 외공 — 활·원거리 무기 계열 |

## 파생 스탯

속성·무예 수치를 기반으로 계산되는 전투 자원.

| 파생 스탯 | 공식 | 비고 |
|-----------|------|------|
| 생명(HP)  | `체질 × 10` | 체질이 유일한 소스 |
| 내력(MP)  | `경맥 × 5 + 내공 × 3` | 경맥=그릇, 내공=쌓인 공력 |
| 체력(AP)  | `의지 × 8` | 스킬 사용 시 소모되는 행동력 |

> **구현 원칙:** 위 공식의 계수(10, 5, 3, 8)는 상수로 분리하여 밸런스 조정 시 코드 변경 없이 수정 가능하게 한다.

---

## 전투에서의 속성 역할

기술 사용 시 **무예 스탯**이 기본 효과를 결정하고, **속성**이 추가 보너스를 제공한다.

| 속성 | 전투 보너스 예시 |
|------|----------------|
| 근력 | 물리 공격 시 데미지 보너스 |
| 민첩 | 연속 공격 기회 제공 |
| 의지 | 반격 판정, 방어 보너스 |
| 경공 | 회피 보너스 |
