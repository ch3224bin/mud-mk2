## 1. 전투 자원 구조

용윤입지전의 전투는 네 가지 독립 자원이 동시에 운용되는 구조다.  
복생(부활) 효과가 생명 · 내력 · 체력 · 가세 · 행동력을 각각 별도로 비율 회복하는 점에서 이 자원들이 완전히 독립되어 있음을 확인할 수 있다.

| 자원 | 한자 | 원천 속성 | 소진 시 패널티 |
|------|------|----------|--------------|
| 생명 | 生命 (HP) | 체질 × 10 + 절기 × 3 | 전투 이탈(사망) |
| 내력 | 内力 (MP) | 경맥 × 5 + 내공 × 3 | 무공 사용 불가 |
| 체력 | 体力 (AP) | 민첩 × 8 | 연격·추가행동 불가, 선제 속도 −30% |
| 가세 | 架勢 | AP와 통합 또는 별도 운영 | 회피 불가 |

> **체력(AP)의 핵심 개념**: HP처럼 0이 되면 사망하는 자원이 아니다.  
> "이 캐릭터가 한 전투에서 얼마나 많은 행동을 폭발적으로 쏟아낼 수 있는가"를 결정하는 **연속 행동력 자원(Stamina)** 이다.

### 체력(AP) vs 가세(架勢)의 구분

| 자원 | 역할 | 소모 행동 |
|------|------|---------|
| 체력(AP) | 공격 행동의 연료 | 연격 발동, 무거운 무공 사용, 위치 이동 |
| 가세(架勢) | 회피 행동의 연료 | 회피 1회당 5 소모, 0이면 회피 불가 |

**MUD 구현 선택지**

- **통합 방안**: AP 하나가 공방 양쪽 연료를 겸함 (구현 단순, 권장)  
  - 회피 시 AP 5 소모, 연격 시 AP 10 소모
- **분리 방안**: AP(공격)와 가세(회피)를 별도 수치로 유지 (원작 충실, 관리 복잡)

## 2. 파생 스탯 공식

### 기본 파생 공식

```
생명(HP) 최대치 = 체질 × 10 + 절기 × 3
내력(MP) 최대치 = 경맥 × 5  + 내공 × 3
체력(AP) 최대치 = 민첩 × 8
```

> 계수는 `CharacterStats`의 `static final` 상수로 관리하므로 밸런스 조정 시 코드 변경 없이 수정 가능.

### 전투 파생 속성 공식

```
선제 속도    = 민첩 × 1.0 + 경공 × 0.5 + 장비 속도 보너스
명중치       = 지력 × 0.8 + 해당외공 × 0.5 + 장비 명중 + 50(기본값)
회피치       = 민첩 × 0.8 + 경공 × 0.4  + 장비 회피
치명타율(%)  = 근력 × 0.3 + 장비 치명타
방어 수치    = 장비 방어  + 의지 × 0.4
방어율(%)    = 장비 방어율 + 의지 × 0.2 + 해당외공 × 0.15   [상한 75%]
연격 발동율(%) = 경맥 × 0.25 + 무공 연격 보너스
단련율(%)    = 근력 × 0.2  + 장비 단련 보너스
반격 발동율(%) = 의지 × 0.3  + 무공 반격 보너스
압제 수치    = 체질 × 0.4  + 장비 압제 보너스
압제율(%)    = 압제 수치 / (압제 수치 + 100) × 100   [연성 캡, 상한 없음]
부상 저항율(%) = 체질 × 0.25 + 의지 × 0.3
```

> **확률 속성 변환 원칙**: 연격 발동율·반격 발동율·압제율 등 "수치 → 확률(%)" 변환이 필요한 속성은  
> 모두 `수치 / (수치 + 100) × 100` 공식을 따른다. 수치가 높을수록 확률이 오르되 100%에 수렴하지 않아 밸런스가 자연 유지된다.

## 4. 전투 판정 흐름

ATB 게이지가 100에 도달한 캐릭터가 행동권을 얻으면 아래 순서로 판정이 진행된다.

```
[ATB 게이지 100 도달 → 행동권 획득]
    │
    ▼
① 명중 판정 ──[회피 성공]──→  "○○이 공격을 피했다!" (AP 5 소모) → 대기
    │ 명중
    ▼
② 기본 피해 계산 ────────────  무기 기본치 × 무예 배율 + 근력 보너스
    │
    ▼
③ 치명타 판정 ───────────────  근력 기반 치명타율 → 피해 × 1.5~2.0
    │
    ▼
④ 방어 적용 ─────────────────  피해 × (1 − 방어율) − 방어수치, 최소 1
    │
    ▼
⑤ 연격 판정 ─────────────────  경맥 기반 연격율, AP 10 소모/타
    │
    ▼
⑥ 반격 판정 (피격자) ────────  의지 기반 반격율, AP 8 소모
    │
    ▼
HP 차감 & 상태이상 확인 → 게이지 리셋 후 충전 재개
```

## 5. ATB 턴 시스템

선제 속도를 실시간 게이지로 운용하는 방식이다.  
각 캐릭터의 게이지가 독립적으로 충전되어, 먼저 100에 도달한 쪽이 행동권을 얻는다.

### 기본 작동 원리

```
각 캐릭터는 개인 ATB 게이지 (0 ~ 100) 를 보유한다.

매 틱(tick)마다:
  게이지 += 선제 속도 / SPEED_DIVISOR

게이지 ≥ 100 → 행동권 획득 → 행동 실행 → 게이지 리셋
```

`SPEED_DIVISOR`는 전체 전투 속도를 조율하는 전역 상수다.  
예) `SPEED_DIVISOR = 10`이면 선제 속도 50인 캐릭터는 2틱마다, 100인 캐릭터는 1틱마다 행동한다.

### 게이지 리셋 방식 — 잔여 유지(권장)

```
행동 후 게이지 = 행동 직전 게이지 − 100

예) 게이지 113에서 행동 → 리셋 후 13 유지
```

잔여를 이월하면 선제 속도 200인 캐릭터가 100인 캐릭터의 정확히 2배로 행동하게 되어 속도 차이가 선형으로 반영된다.

### 상태이상과 ATB의 연계

지속시간을 "턴" 대신 **틱(tick)** 단위로 정의하면 ATB와 자연스럽게 연동된다.

```
현운(眩暈):
  ATB 게이지 충전은 계속되지만,
  게이지 100 도달 시 행동권을 소멸시키고 지속틱을 −1 차감한다.
  → 빠른 캐릭터일수록 현운 피해가 크다.

압제(壓制):
  ATB 게이지 충전 속도 자체를 ×0.5로 감소시킨다.
  → 빠른 캐릭터도 압제 중에는 느려진다.
```

### MUD 텍스트 출력 방안

**행동 시 자동 출력 (방안 A — 권장)**
```
[전투] 홍길동의 게이지가 찼다. (홍길동 100 | 적장 67 | 부하A 45)
홍길동이 청풍검법을 시전한다!
```

**`status` 명령어 조회 (방안 B — 선택)**
```
> status
┌─ ATB 게이지 ───────────────┐
│ 홍길동   ████████░░  82%   │
│ 적장     █████░░░░░  54%   │
│ 부하A    ███████░░░  70%   │
└────────────────────────────┘
```

## 6. ① 선제권 / ATB 게이지

```
선제 속도 = 민첩 × 1.0 + 경공 × 0.5 + 장비 속도 보너스
```

- **민첩(속성)**: ATB 게이지 충전 속도의 주 기여자 — "얼마나 빨리 행동하는가"
- **경공(무예)**: 보조 기여자 — 경공이 높을수록 연속 행동 간격이 좁아짐
- 게이지 동시 도달 시 민첩 수치가 더 높은 쪽 우선

## 7. ② 명중 / 회피 판정

명중이 회피를 초과하지 못하면 타격이 성립하지 않는다.

```
명중치 = 지력 × 0.8 + 해당외공 × 0.5 + 장비 명중 + 50

회피치 = 민첩 × 0.8 + 경공 × 0.4 + 장비 회피

회피율(%) = clamp(회피치 − 명중치, 0, 75)

판정:
  roll = random(1, 100)
  roll < 회피율  →  회피 성공 (AP 5 소모 or 가세 5 소모)
  roll ≥ 회피율  →  명중
```

> 최대 회피율 75% 캡 적용 — "전체 회피는 불가능"한 원작 철학 반영.  
> AP(가세)가 0이 되면 회피 판정 자체를 시도하지 못한다.

## 8. ③ 기본 피해 계산

```
무예 배율  = 해당 외공 수치 × 0.008
             (예: 외공 60 → 배율 1.48)

기본 피해  = 무기 기본치 × (1 + 무예 배율) + 근력 × 0.3

피해 변동  = 기본 피해 × random(0.9, 1.1)    ← ±10% 랜덤
```

### 무기 기본치 참고 범위 (등급별)

| 무공 등급 | 색상 | 무기 기본치 범위 | 비고 |
|----------|------|----------------|------|
| 기초 | 灰 (회색) | 5 ~ 15 | 초반 전용 |
| 진계 | 綠 (녹색) | 12 ~ 25 | |
| 상승 | 藍 (청색) | 22 ~ 40 | |
| 비전 | 紫 (자색) | 35 ~ 60 | |
| 정급 | 橙 (주황) | 55 ~ 90 | |
| 절세 | 紅 (적색) | 80 ~ 150 | 만렙 10 이후 |

## 9. ④ 치명타 판정

```
치명타율(%)  = 근력 × 0.3 + 장비 치명타 보너스
치명타 배율  = 1.5  (기본)
               최대 2.0  (천부 또는 특수 무공으로 상승)

발동 시: 최종 피해 × 치명타 배율
```

## 10. ⑤ 방어 적용

호갑 수치(護甲)와 호갑율(護甲率)은 독립된 속성으로 이중 계산된다.

```
방어 수치(armor)     = 장비 방어 + 의지 × 0.4
방어율(%)(armorPct)  = 장비 방어율 + 의지 × 0.2 + 해당외공 × 0.15
                      [상한: 75%]

실질 피해 계산:
  피해1       = 공격 피해 × (1 − armorPct / 100)
  최종 피해   = max(피해1 − armor, 1)
```

### 진짜 피해(真傷 / True Damage)

방어율과 방어 수치를 모두 무시하고 HP를 직접 차감한다.  
상위 빌드(진상류)의 핵심 속성으로, 특정 무공 효과나 천부로만 발동된다.

```
진짜 피해: armor·armorPct 무시 → HP 직접 차감
```

## 11. ⑥ 연격 시스템

한 번의 행동에서 추가 타격이 연쇄 발동되는 메커니즘.  
단련(斷連) 속성이 없는 적은 연격을 차단하지 못한다.

```
연격 발동율(%)  = 경맥 × 0.25 + 무공 연격 보너스

연격 발동 시:
  추가 타격 횟수  = 1d3  (1~3회)
  추가 타격 피해  = 기본 피해의 50%
  AP 소모         = 타격 1회당 10

단련(斷連) 판정:
  방어자 단련율(%) = 근력 × 0.2 + 장비 단련 보너스
  차단율(%)        = clamp(단련율 − 연격율, 0, 75)

  roll = random(1, 100)
  roll < 차단율  →  단련 성공 (연격 차단)
  roll ≥ 차단율  →  연격 관통, 추가 타격 적용
```

> 단련율 = 연격율인 경우 차단율 0% → 대등하면 연격이 유리하다.  
> 단련율이 연격율보다 크더라도 최대 75%만 보장되어 연격류가 완전히 봉인되지 않는다.  
> AP가 소진되면 연격 발동 불가. 연격류 빌드는 急速(급속) · 蓄力(축력) · 回体(회체) 장비 어록으로 AP를 유지하는 것이 핵심.

## 12. ⑦ 반격 시스템

반격은 피격 직후 즉각 발동되는 능동 반응이므로 AP를 소모한다.  
AP가 부족하면 반격 판정 자체가 시도되지 않는다.

```
반격 발동 조건:
  1. 피격 이벤트 발생
  2. random(1, 100) < 반격 발동율(%)
  3. 현재 AP ≥ 반격 AP 소모량      ← AP 부족 시 발동 불가

반격 발동율(%)  = 의지 × 0.3 + 무공 반격 보너스
반격 AP 소모량  = 8
반격 피해       = 기본 피해의 60%
```

> 연격에 시달려 AP가 고갈되면 반격 기회도 사라진다.  
> 전투 후반 AP 관리가 공방 양쪽 모두에 영향을 미치는 구조.

---

## 13. ⑧ 압제 시스템

공격 성공 시 확률적으로 상대의 ATB 게이지 충전을 지연시킨다.  
장병(長兵) 계열이 대표적인 압제 빌드 무기.

```
압제 수치    = 체질 × 0.4 + 장비 압제 보너스
압제율(%)    = 압제 수치 / (압제 수치 + 100) × 100

압제 판정:
  공격 성공 시:
    roll = random(1, 100)
    roll < 압제율  →  [압제] 부여
  [압제] 효과:  ATB 게이지 충전 속도 ×0.5 (15틱 지속)

수치 → 확률 변환 예시:
  압제 수치  50  → 압제율 ≈ 33%
  압제 수치 100  → 압제율 = 50%
  압제 수치 200  → 압제율 ≈ 67%

부상 저항(상태이상 지속 단축):
  부상 저항율(%) = 체질 × 0.25 + 의지 × 0.3
```

---

## 14. 체력(AP) 상세 메커니즘

체력(体力)은 HP가 아닌 **연속 행동력 자원(Stamina)** 이다.  
민첩(灵巧)이 속도 · 회피 · **체력 상한** 세 가지를 동시에 올린다.

### 소모 규칙

| 행동 | AP 소모량 |
|------|---------|
| 연격 추가 타격 1회 | 10 |
| 중급 이상 무거운 무공 사용 | 5 |
| 위치 이동 (전술적 이동) | 3 |
| 회피 1회 (가세 통합 시) | 5 |

### 회복 규칙

| 회복 수단 | 회복량 | 타이밍 |
|----------|--------|--------|
| 행동권 획득 시 자동 회복 | 민첩 × 0.5 | ATB 게이지 100 도달 시마다 |
| 調息(조식, 1행동 소모) | 민첩 × 2 | 즉시 적용 |
| 急速(급속) 장비 어록 | 회복량 % 추가 | 행동권 획득 시 보너스 |

> AP와 MP 모두 **행동권 획득 시** 회복으로 통일한다.  
> 틱마다 회복 시 빠른 캐릭터가 느린 캐릭터보다 훨씬 많이 회복하는 속도 스탯 과부하 문제를 방지한다.

### AP 소진 시 패널티

```
AP ≤ 0:
  - 연격 발동 불가
  - 중급 이상 무공 사용 불가 (기본 공격만 가능)
  - 선제 속도 −30% ([체력 소진] 상태이상)
```

### 체력 관련 장비 词條(어록) 분류

| 어록 | 효과 |
|------|------|
| 急速 (급속) | 턴 시작 AP 회복량 증가 |
| 蓄力 (축력) | 대기 시 AP 회복량 증가 |
| 回体 (회체) | 행동 후 즉시 AP 일부 환원 |
| 護足 (호족) | 가세 회복 (회피 분리 운영 시) |

## 15. 상태이상 상세 정의

상태이상은 성격에 따라 세 가지로 분류하여 각각 다른 지속·해제 규칙을 적용한다.

### 분류 기준

| 분류 | 특성 | 해당 상태이상 |
|------|------|------------|
| A. 틱 소멸형 | ATB 틱 경과 시 자동 해소 | 현운, 압제 |
| B. 누적·자연회복형 | 수치가 쌓이고 전투 외에서 서서히 감소 | 독, 내상 |
| C. 치료 필수형 | 시간 경과로는 회복 불가, 처치 필요 | 외상, 점혈 |

---

### A. 틱 소멸형

#### 현운(眩暈) — 완전 행동 불가

전투 중 강력한 제어기. 지속 중 ATB 게이지는 충전되지만 행동권이 자동 소멸된다.

```
지속시간     = 20틱
효과         = ATB 게이지 100 도달 시 행동권 소멸 (게이지는 0으로 리셋)
해제         = 지속틱 경과 후 자동 해제
저항 적용    = 부상 저항율(%)만큼 지속시간 단축
               예) 저항율 30% → 실제 지속 14틱
중첩         = 새 현운 부여 시 남은 틱 수와 비교하여 더 큰 값으로 갱신
```

#### 압제(壓制) — ATB 충전 속도 감소

```
지속시간     = 15틱
효과         = ATB 게이지 충전 속도 ×0.5
해제         = 지속틱 경과 후 자동 해제
저항 적용    = 부상 저항율(%)만큼 지속시간 단축
중첩         = 현운과 동일, 더 큰 값으로 갱신
```

---

### B. 누적·자연회복형

전투 중에는 자연 회복이 없다. 전투 외 상황에서 시간이 지나면 서서히 감소한다.  
약은 즉각 해결 수단, 시간은 무료지만 느린 수단이다.

#### 독(毒) — 매 틱 HP 지속 감소

```
부여 시      = 독 스택 +1 (최대 10스택까지 누적)
틱당 피해    = 스택 × 3  (진짜 피해, 방어 무시)
전투 중      = 자연 회복 없음
전투 외      = 매 분(실시간) 스택 −1 자동 감소
즉시 해제    = 해독약(解毒藥) 사용 → 전체 스택 제거
저항 적용    = 부상 저항율(%)로 스택 부여 자체를 확률적으로 차단
```

#### 내상(内傷) — 매 틱 MP 지속 감소

```
부여 시      = 내상 수치 + [무공 내상 수치]만큼 누적
틱당 효과    = MP − (내상 수치 × 0.5)
전투 중      = 자연 회복 없음
전투 외      = 매 분 내상 수치 −5 자동 감소
즉시 해제    = 회기단(回氣丹) 사용, 또는 조식(調息) 10틱 지속 시 전량 초기화
저항 적용    = 부상 저항율(%)로 내상 수치 부여량 감소
```

---

### C. 치료 필수형

시간이 지나도 자연 회복되지 않는다. 반드시 처치가 필요하다.  
방치 시 전투 돌입 때마다 패널티 상태로 진입한다.

#### 외상(外傷) — 특정 속성 감소

```
효과         = 부여 무공에 따라 지정된 속성 −[외상 수치]%
               (예: 근력 외상 → 근력 감소, 민첩 외상 → 민첩·속도 감소)
자연 회복    = 없음
해제 방법:
  ① 치상약(治傷藥) 사용  → 즉시 해제
  ② 의원(醫員) 방문      → 즉시 해제
  ③ 조식(調息) 장기 유지 → 30틱마다 외상 수치 −10  (느리지만 무료)
```

#### 점혈(點穴) — 공격 행동 불가

```
효과         = 공격·무공 사용 불가 (이동·회피·반격은 가능)
자연 회복    = 없음
해제 방법:
  ① 통맥환(通脈丸) 사용              → 즉시 해제
  ② 아군의 해혈(解穴) 무공 시전     → 즉시 해제
  ③ 자가 해혈 시도 (권장 보유 시)   → 성공률 = 권장 수치 × 0.5%
```

---

### 전체 비교 요약

| 상태이상 | 분류 | 전투 중 자연 회복 | 전투 외 자연 회복 | 약으로 즉시 해제 | 비고 |
|---------|------|:--------------:|:--------------:|:--------------:|------|
| 현운 眩暈 | 틱 소멸 | ✓ 20틱 후 | — | — | 빠를수록 피해 큼 |
| 압제 壓制 | 틱 소멸 | ✓ 15틱 후 | — | — | ATB 충전 감속 |
| 독 毒 | 누적·자연 | ✗ | ✓ 서서히 | 해독약 | 방어 무시 피해 |
| 내상 内傷 | 누적·자연 | ✗ | ✓ 서서히 | 회기단 | MP 고갈 압박 |
| 외상 外傷 | 치료 필수 | ✗ | ✗ | 치상약/의원 | 속성 감소 지속 |
| 점혈 點穴 | 치료 필수 | ✗ | ✗ | 통맥환/해혈 | 아군 협력으로 해제 가능 |

### 상태이상 저항 판정 (공통)

```
저항율(%) = 체질 × 0.25 + 의지 × 0.3

부여 시도마다:
  roll = random(1, 100)
  roll < 저항율  →  틱 소멸형: 지속시간 단축 / 누적형: 부여량 감소 / 치료필수형: 면역
  roll ≥ 저항율  →  정상 부여
```

## 16. 내력(MP) 소모와 회복

```
무공 사용 시 MP 소모  = 무공 등급 계수 × 기본 소모량

무공 등급 계수:
  기초(灰) 1.0 / 진계(綠) 1.2 / 상승(藍) 1.5
  비전(紫) 2.0 / 정급(橙) 2.8 / 절세(紅) 4.0

행동권 획득 시 자동 회복 (ATB 게이지 100 도달 시마다):
  MP += 경맥 × 0.3
  AP += 민첩 × 0.5

調息(조식) 회복     = 내공 × 0.8  (전투 외 1행동 소모, 즉시 적용)
回内 장비 어록      = 행동권 획득 시 추가 MP 회복 %
```

> **회복 타이밍 근거**: 틱마다 회복하면 ATB 속도가 빠른 캐릭터가 느린 캐릭터보다 훨씬 많이 회복해  
> 선제 속도 스탯의 가치가 과도하게 커진다. 행동권 획득 시 1회 회복으로 통일하면  
> 느린 캐릭터도 자기 차례에 동일한 회복을 받아 밸런스가 안정된다.

## 17. 무공 시스템

무공(武功)은 전투 스타일을 결정하는 핵심 요소다. 캐릭터는 배운 무공 중 일부를 **장착(裝備)** 해야 전투에서 사용할 수 있으며, 장착하지 않은 무공은 속성 기여만 남긴다.

### 무공의 네 종류와 장착 슬롯

| 종류 | 한자 | 역할 | 장착 슬롯 |
|------|------|------|---------|
| 내공 | 内功 | 상시 패시브 — MP 상한, 연격율, 특수 코어 속성 부여 | **1개 고정** |
| 경공 | 輕功 | 상시 패시브 — ATB 속도, 회피, 이동 | **1개 고정** |
| 절기 | 絶技 | 상시 패시브 — HP 상한, 반격, 압제 등 보조 속성 | **1개 고정** |
| 외공 | 外功 | 전투 중 능동 기술 — 실제 공격 초식 | **가변 (초기 2개 → 최대 8개)** |

내공·경공·절기는 장착하면 효과가 상시 발동되며, 돌파 효과도 장착 중에만 활성화된다.  
외공은 장착한 초식만 전투 중 사용할 수 있다. 같은 종류(검법, 도법 등)를 여러 개 장착해 CD를 돌아가며 쓰는 것이 핵심 운용법이다.

---

### 내공(内功) — 코어 빌드를 결정하는 1슬롯

내공은 빌드 전체의 방향성을 결정한다. 어떤 내공을 장착하느냐에 따라 캐릭터의 전투 테마가 달라진다.

**원작 주요 내공 예시**

| 내공명 | 등급 | 코어 속성 | 빌드 방향 |
|--------|------|---------|---------|
| 易筋經 (역근경) | 절세 | 무적 | 1턴 무적 발동류 |
| 洗髓經 (세수경) | 절세 | 현운 | 현운 축적 제어류 |
| 萬惡魔功 (만악마공) | 절세 | 내상 | 내상 누적 내력 소진류 |
| 歸墟秘典 (귀허비전) | 절세 | 흡내 | 적 내력 흡수 · 반사류 |
| 三洞靈寶經 (삼동영보경) | 절세 | 진상 | 방어 무시 진짜 피해류 |
| 龍胤歸藏功 (용윤귀장공) | 절세 | 6속성 +10 | 전 속성 균형 육각형류 |

**MUD 내공 설계 예시 (한국풍)**

```
[현무신공 玄武神功] — 정급(橙)
  장착 효과: 경맥 +8, 방어율 +5%, 매 틱 HP 회복 +경맥×0.1
  돌파 효과: 피격 시 20% 확률로 [철갑] 발동 → 다음 공격 피해 흡수
  코어 속성: 철갑(무적) / 방어 강화류

[천뇌공 天雷功] — 비전(紫)
  장착 효과: 지력 +6, 명중 +10, 연격 발동 시 번개 외상 추가 부여
  돌파 효과: 외공 사용 시 5% 확률로 현운 부여
  코어 속성: 현운 / 제어류

[오독심공 五毒心功] — 비전(紫)
  장착 효과: 지력 +5, 독 스택 최대치 +3, 독 틱 피해 +20%
  돌파 효과: 공격 성공 시 독 스택 자동 +1
  코어 속성: 독 / 누적 데미지류
```

---

### 경공(輕功) — 속도와 생존을 담당하는 1슬롯

경공은 ATB 속도와 회피에 직접 기여하며, 체력(AP) 효율에도 영향을 준다.

**원작 주요 경공 예시**

| 경공명 | 등급 | 주요 효과 |
|--------|------|---------|
| 天人感應步 (천인감응보) | 절세 | 영교(민첩)·회피 대폭 강화 |
| 凌虛飛渡 (능허비도) | 정급 | 회피 특화, 飄逸 버프 부여 |
| 青烟飛絮功 (청연비서공) | 정급 | 회피+ATB 속도 균형형 |

**MUD 경공 설계 예시**

```
[쾌풍신법 快風身法] — 진계(綠)
  장착 효과: 민첩 +4, 선제 속도 +5
  돌파 효과: 회피 성공 시 AP 소모량 −2 (최소 1)
  특징: 초반 속도 확보용

[능운보 凌雲步] — 비전(紫)
  장착 효과: 민첩 +7, 경공 +6, 회피 +8%
  돌파 효과: AP 최대치 +민첩×2 추가, 회피 성공 시 다음 공격 치명타율 +20%
  특징: 회피-치명타 연계 시너지

[철족보 鐵足步] — 상승(藍)
  장착 효과: 민첩 +5, 체질 +3
  돌파 효과: 매 틱 가세 +3 자동 회복
  특징: 가세 유지 특화, 회피 지속성 확보
```

---

### 절기(絶技) — 보조 강화를 담당하는 1슬롯

절기는 HP 상한에 기여하며, 반격·압제·상태이상 강화 등 다양한 보조 효과를 제공한다.

**원작 주요 절기 예시**

| 절기명 | 등급 | 주요 효과 |
|--------|------|---------|
| 金剛不壞神功 (금강불괴신공) | 절세 | 무적 강화, 절기 최고 방어형 |
| 道德眞經 (도덕진경) | 절세 | 진상 강화, 내상 조합 |
| 斗轉星移 (두전성이) | 정급 | 피해 반사, 반격 강화 |
| 虎嘯龍吟 (호소용음) | 정급 | 점혈 확률 강화 |

**MUD 절기 설계 예시**

```
[철신공 鐵身功] — 상승(藍)
  장착 효과: 절기 +5, 체질 +4, 방어 수치 +8
  돌파 효과: 피격 피해가 최대 HP의 20% 초과 시 초과분 50% 감소
  특징: 초반 탱킹 기반

[팔황장 八荒掌] — 비전(紫)
  장착 효과: 절기 +6, 근력 +5, 압제율 +10%
  돌파 효과: 압제 부여 성공 시 대상의 ATB 게이지 −20 추가
  특징: 압제 강화 제어형

[음양조화공 陰陽調和功] — 정급(橙)
  장착 효과: 절기 +8, 경맥 +6, 매 틱 MP 회복 +내공×0.2
  돌파 효과: HP 50% 이하 시 반격율 +20%
  특징: 내력 유지 + 역전 반격형
```

### 외공(外功) — 전투 능동기, 가변 슬롯

외공은 실제 전투에서 사용하는 초식이다. 슬롯 수가 늘어날수록 다양한 초식을 돌아가며 쓸 수 있어 AP·MP 관리 효율이 올라간다.

#### 외공 슬롯 수 증가 — MUD 설계안

원작에서 슬롯 증가의 정확한 기준은 공개되지 않았다. MUD에서는 **캐릭터 숙련도(무공 레벨 합산)** 를 기준으로 단계적으로 해금하는 방식을 제안한다.

| 단계 | 외공 슬롯 수 | 해금 조건 (예시) |
|------|:-----------:|----------------|
| 초입 | 2개 | 시작 시 기본 제공 |
| 진계 | 3개 | 외공 무예 합산 30 이상 |
| 상승 | 4개 | 외공 무예 합산 60 이상 |
| 비전 | 5개 | 외공 무예 합산 100 이상 |
| 정급 | 6개 | 외공 무예 합산 150 이상 |
| 절세 | 8개 | 외공 무예 합산 200 이상 (상한) |

> 외공 무예 합산 = 장착 중인 외공 종류(검법·도법·권장 등)의 무예 수치 합계.  
> 많은 무공을 익혀 전문화할수록 더 많은 초식을 쓸 수 있다는 성장 논리와 일치한다.

#### 외공 초식 구조

각 외공 초식은 다음 요소로 구성된다.

```
초식 구성:
  무기 유형    = 검법 / 도법 / 권장 / 장병 / 기문 / 사술 중 1
  기본 피해    = 초식별 고유 수치
  MP 소모      = 등급 계수 × 기본 소모량
  AP 소모      = 초식 무게 (경형 3 / 중형 5 / 중장형 8)
  쿨다운(CD)   = 동일 초식 연속 사용 불가 틱 수
  특수 효과    = 돌파 시 해금되는 추가 효과
```

**MUD 외공 초식 설계 예시**

```
[청풍검법 1초식 — 청풍일격 淸風一擊]
  종류: 검법 / 등급: 상승(藍)
  기본 피해: 무기치 × 1.3
  MP 소모: 12 / AP 소모: 3 (경형)
  CD: 0틱 (기본기, 쿨다운 없음)
  돌파 효과: 명중 시 연격율 +15%

[추혼탈명검 追魂奪命劍]
  종류: 검법 / 등급: 비전(紫)
  기본 피해: 무기치 × 1.8
  MP 소모: 22 / AP 소모: 5 (중형)
  CD: 8틱
  돌파 효과: 연격 발동 시 추가 타격 피해 +20%
  특수: 연격류 핵심 초식

[강산만리창 江山萬里槍]
  종류: 장병 / 등급: 비전(紫)
  기본 피해: 무기치 × 2.0
  MP 소모: 28 / AP 소모: 8 (중장형)
  CD: 12틱
  돌파 효과: 명중 시 압제 부여 확률 +25%
  특수: 범위 공격 (전방 2칸 적에게 동시 적중 가능)

[만화귀종장 萬花歸宗掌]
  종류: 권장 / 등급: 정급(橙)
  기본 피해: 무기치 × 1.5 (다단 3회)
  MP 소모: 35 / AP 소모: 15 (연속 소모)
  CD: 15틱
  돌파 효과: 각 타격마다 독 스택 +1 부여
  특수: 3연타 초식, 각 타격에 명중 판정 별도 적용

[암기삼연발 暗器三連發]
  종류: 기문 / 등급: 상승(藍)
  기본 피해: 무기치 × 0.8 (3연속 발사)
  MP 소모: 18 / AP 소모: 5
  CD: 6틱
  돌파 효과: 명중한 적에게 점혈 확률 +10%
  특수: 원거리 공격, 명중 판정에 지력 보너스 2배 적용
```

### 무공 장착 빌드 예시

실제로 내공·경공·절기·외공이 어떻게 조합되는지 예시를 보여준다.

#### 연격-검법 빌드

```
내공: 귀허비전 歸墟秘典 (흡내 · 내상)
  → 매 틱 적 MP 흡수, 내상 부여로 MP 고갈 압박

경공: 능운보 凌雲步 (회피 · AP 보강)
  → 회피 성공 후 치명타율 상승, 연격 후 AP 효율↑

절기: 음양조화공 陰陽調和功 (MP 유지 · 역전 반격)
  → 내공의 MP 소모를 절기가 회복으로 보조

외공 (4슬롯 예시):
  [청풍일격]     — 기본기, CD 없음, 연격율 부여
  [추혼탈명검]   — 연격 강화 핵심 초식
  [현허검기 玄虛劍氣] — 내상 부여 초식 (내공 코어와 시너지)
  [천풍십삼검 天風十三劍] — 고피해 마무리기
```

#### 압제-장병 빌드

```
내공: 태극신공 太極神功 (내상 · 속도)
경공: 철족보 鐵足步 (가세 유지)
절기: 팔황장 八荒掌 (압제 강화)

외공 (3슬롯 예시):
  [강산만리창]   — 압제 부여 핵심
  [반월대도참 半月大刀斬] — 광역 도법 초식
  [일기당천봉 一氣當千鋒] — 장병 고피해 단타
```

## 18. 무공 등급과 성장

무공은 10레벨 만렙, 6단계 등급이 존재한다.  
등급이 높을수록 경험치 요구량이 급격히 증가한다.

| 등급 | 색상 | 학습 상한 | 피해 배율 | 비고 |
|------|------|---------|---------|------|
| 기초 | 灰 회색 | 12개 | ×1.0 | 초반 속성 올리기 전용 |
| 진계 | 綠 녹색 | 10개 | ×1.2 | |
| 상승 | 藍 청색 | 8개 | ×1.5 | |
| 비전 | 紫 자색 | 6개 | ×2.0 | 잠재력 상한 상승 시작 |
| 정급 | 橙 주황 | 4개 | ×2.8 | 주전 무공 구간 |
| 절세 | 紅 적색 | 2개 | ×4.0 | 만렙 후 돌파 효과 해금 |

> 상한 초과 학습 시 경험치 획득 −50% 패널티.

### 突破(돌파) 효과 — 만렙(10) 이후 해금

| 돌파 종류 | 효과 예시 |
|----------|---------|
| 내공/경공/절기 (상시) | 연격율 +%, 회피 +%, 반격 +% 등 영구 적용 |
| 외공/무기초식 (사용 시) | 범위 피해 해금, 다단 공격, 부여 상태이상 추가 |
| 품질별 배율 | 백색(×1) → 녹색(×2) → 적색(×10) |

## 설계 우선순위 로드맵

| 단계 | 구현 항목 | 이유 |
|------|----------|------|
| 1 | HP / MP / AP 세 자원 분리 | 전투의 기반 구조 |
| 2 | ATB 게이지 + 틱 루프 | 턴 순서 결정의 핵심 엔진 |
| 3 | 명중 / 회피 판정 + AP 소모 | 회피가 무한하지 않게 만드는 핵심 제약 |
| 4 | 이중 방어 (방어율 % + 방어 수치) | 원작 호갑 구조 충실 재현 |
| 5 | 내공·경공·절기 1슬롯 고정 장착 | 빌드 테마 결정 구조 |
| 6 | 외공 슬롯 시스템 + 초식 CD | 능동 전투의 핵심 |
| 7 | 연격 + 단련 + AP 소진 | 공방 자원 관리의 핵심 |
| 8 | 반격 + AP 소모 조건 | AP 고갈의 전략적 의미 완성 |
| 9 | 상태이상 6종 + 분류별 해제 규칙 | 전략 다양성 및 전투 외 콘텐츠 연결 |
| 10 | 진짜 피해(真傷) | 상위 빌드 차별화 |
| 11 | 무공 등급 배율 + 돌파 효과 | 장기 성장 동기 부여 |

## 19. 용어집

문서 전반에서 사용하는 주요 용어의 한자 원어·의미·코드 식별자를 한곳에 정리한다.

### 전투 자원

| 용어 | 한자 | 코드 식별자 | 의미 |
|------|------|-----------|------|
| 생명 | 生命 | `hp` | 0이 되면 전투 이탈. 체질·절기에서 파생 |
| 내력 | 内力 | `mp` | 무공 사용 연료. 경맥·내공에서 파생 |
| 체력 | 体力 | `ap` | 연속 행동 연료(Stamina). 민첩에서 파생. 0이 돼도 사망하지 않고 행동 제한 |

### 전투 판정 속성

| 용어 | 한자 | 코드 식별자 | 의미 |
|------|------|-----------|------|
| 선제 속도 | 先制速度 | `initiativeSpeed` | ATB 게이지 충전 속도 |
| 명중 | 命中 | `accuracy` | 공격이 상대에게 닿을 확률 기여값 |
| 회피 | 回避 | `evasion` | 공격을 피할 확률 기여값 |
| 치명타율 | 暴擊率 | `critRate` | 치명타 발동 확률 (%) |
| 치명타 배율 | 暴擊倍率 | `critMultiplier` | 치명타 발동 시 피해 배수 |
| 방어 수치 | 護甲 | `armor` | 피해에서 정수로 차감되는 방어값 |
| 방어율 | 護甲率 | `armorPct` | 피해를 % 단위로 감소시키는 방어값 (상한 75%) |
| 연격율 | 連擊率 | `comboRate` | 추가 타격 발동 확률 (%) |
| 단련율 | 斷練率 | `comboBreakRate` | 상대 연격을 차단할 확률 기여값 |
| 반격율 | 反擊率 | `counterRate` | 피격 후 즉시 반격 발동 확률 (%) |
| 압제 수치 | 壓制値 | `suppressionPower` | 압제율(%) 변환 전 날 것의 수치 |
| 압제율 | 壓制率 | `suppressionRate` | 공격 성공 시 [위압] 부여 확률 (%) |
| 부상 저항율 | 負傷抵抗率 | `injuryResist` | 상태이상 면역·지속시간 단축 확률 (%) |
| 진상 | 真傷 | `trueDamage` | 방어 수치·방어율을 무시하는 피해 유형 |

### 행동 및 회복

| 용어 | 한자 | 코드 식별자 | 의미 |
|------|------|-----------|------|
| 조식 | 調息 | `meditate` | 숨을 고르는 회복 행동. 1행동 소모, MP·AP 즉시 회복. 전투 외 사용 |

### 상태이상

| 용어 | 한자 | 코드 식별자 | 분류 | 의미 |
|------|------|-----------|------|------|
| 혼절 | 昏絶 | `STUNNED` | 틱 소멸 | 20틱간 행동권 소멸. 원어: 현운(眩暈) |
| 위압 | 威壓 | `SUPPRESSED` | 틱 소멸 | 15틱간 ATB 충전 속도 ×0.5. 원어: 압제(壓制) |
| 독 | 毒 | `POISONED` | 누적·자연회복 | 매 틱 HP 감소(방어 무시). 스택 누적 |
| 내상 | 内傷 | `INTERNAL_INJURY` | 누적·자연회복 | 매 틱 MP 감소. 수치 누적 |
| 외상 | 外傷 | `EXTERNAL_INJURY` | 치료 필수 | 특정 속성 % 감소. 자연 회복 없음 |
| 점혈 | 點穴 | `ACUPOINT_SEALED` | 치료 필수 | 공격·무공 사용 불가. 자연 회복 없음 |

### 장비 어록(词條) — AP·MP 회복 관련

| 용어 | 한자 | 코드 식별자 | 의미 |
|------|------|-----------|------|
| 쾌회 | 快回 | `QUICK_RECOVERY` | 행동권 획득 시 AP·MP 회복량 % 증가. 원어: 급속(急速) |
| 축기 | 蓄氣 | `ACCUMULATE_QI` | 조식(대기) 시 AP·MP 회복량 % 증가. 원어: 축력(蓄力) |
| 회력 | 回力 | `RETURN_FORCE` | 행동 직후 소모 AP의 일부 즉시 환원. 원어: 회체(回體) |
| 수세 | 守勢 | `GUARD_STANCE` | 매 행동권 획득 시 기세(AP) 추가 회복. 원어: 호족(護足) |

### ATB 시스템

| 용어 | 코드 식별자 | 의미 |
|------|-----------|------|
| ATB 게이지 | `atbGauge` | 0~100 범위. 100 도달 시 행동권 획득 |
| 틱 | `tick` | ATB 게이지가 한 번 갱신되는 최소 시간 단위 |
| 속도 분모 | `SPEED_DIVISOR` | `static final` 상수. 전체 전투 속도 조율 |
| 행동권 | — | ATB 게이지 100 도달 시 부여되는 1회 행동 기회 |
